# OG image fonts

Static TTF instances used by `lib/og/template.tsx` to render Open Graph and
Twitter images with `next/og` (Satori does not support variable fonts, so the
variable fonts loaded via `next/font` in the app cannot be reused here).

- `bricolage-grotesque-*.ttf` — [Bricolage Grotesque](https://fonts.google.com/specimen/Bricolage+Grotesque), SIL Open Font License 1.1
- `geist-mono-500.ttf` — [Geist Mono](https://fonts.google.com/specimen/Geist+Mono), SIL Open Font License 1.1

Both were downloaded from Google Fonts as static per-weight instances.
